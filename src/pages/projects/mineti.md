# Mineti - Timesheet tracking application

Mineti is a timesheet tracking application that allows users to track their time spent on different tasks and projects. It is a simple and easy to use application that can be used by individuals or teams to track their time and get a better understanding of their productivity.

## Features

- Timesheet tracking
- Time tracking
- Time reporting
- Time analysis
- Time visualization

## Tech stack

- Tauri
- Vue 3
- Vite
- VitePress
- Tailwind CSS
- Pinia
